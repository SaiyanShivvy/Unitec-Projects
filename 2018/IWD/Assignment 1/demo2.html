HTML DOM Range Object

HTML - Setup

First we need to create an <input> with the type"range"and with an ID "slider".
For this example, I am using an ID "demoText" for the paragraph as I am only wanting to change the font-size of a certain <p>.

      <input type="range" min="10" max="48" value="14" id="slider"/>

We can also give the <input> a "min" & "max" value, otherwise by default it would be 0 to 100. We can also specify a "value" this is the default value of the "range" object.

JavaScript - Setup

First, we need to establish some variables.
The load handler will do the main setup once the page is fully loaded. In this example we are also specifying the default value for the <input>.
These will go in a <script> tag, for this example.

      var slider;

When the <input> receives an input, the function will trigger.
The code inside the function get the "value" and assigns it to a var called slider and takes that and sets it to the <p> tags style.fontSize. The + "px" is to specify the units, which in the case of fontSize in this example is pixels. 

      window.addEventListener('input', function() {                            
       slider = document.getElementById("slider").value;                 
       document.getElementById("demoText").style.fontSize = slider + "px"
       }, false);
