HTML DOM Select Object

HTML - Setup

For this example we need to create an <select> object with an ID.
Create some options using the <option> tags give it a "value". 

			<select id="fonts" onchange="run()">			
			<option value="Georgia, serif">	Georgia<option>
				     </select>

The onchange option is used to call a JavaScript funtion.
In this example, I am using custom font with you will need to import either using the standard <link href="https://fonts.googleapis.com/css?family=fontname" rel="stylesheet"> or using import,
@import url('https://fonts.googleapis.com/css?family=fontname') in the <head> tags.
A good website to get font from is https://fonts.google.com/

JavaScript - Setup

For the JavaScript, we need to create a function called "run()". This will be called whenever the selected option of the <select> element is changed.
Finally, add the following line to change the font:
document.getElementById("demoText").style.fontFamily = document.getElementById("font").value;
