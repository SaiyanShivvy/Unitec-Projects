HTML DOM Input Color Object

<input> elements of type "color" provide a user interface element that allows a user specify a color by, using a color picker interface or an text field allowing the user to type in the hex value e.g."#0000FF".

Only simple colors (with no alpha channel) are allowed. The values are compatible with CSS.

HTML - Setup

The HTML is quite simple, first we need to create an <input> with the type"color"and with an ID "Color", this will be used to change the color of the paragraph's text.
For this example I am using an ID "para" for the paragraph as I am only wanting to change the color of a certain <p>.

JavaScript - Setup

First, we need to establish some variables.
The load handler will do the main setup once the page is fully loaded. In this example we are also specifying the default value for the <input>.
These will go in a <script> tag, for this example.

      var color;                                      
      var colorVal = "#0000ff";                       
      var paraColor;                                  
      window.addEventListener("load", startup, false);

When the page is loaded, the load will call the startup() function.

      function                                             
       color = document.querySelector("#colorS");          
       color.value = colorVal;                             
       color.addEventListener("change", update, false);

After a color is chosen and the color picker is closed.
A "change" event is sent to the element.
This "change" event will trigger the update() function.
The event.target.value will get the value of the color and assign it to the paragraphs css, which is accessed with the paraColor.style.color code.
querySelector() can be used to find elements using ID or tag, so #para can be replaced with p to select an element with a <p> tag. 

      function update(event) {                             
       paraColor = document.querySelector("#para");        
       paraColor.style.color = event.target.value;         
      }
